# Factor Dossiers

These files are the canonical deep-context layer for factor-specific questions.

Use them after:

1. `docs/agent-context/factors-reference.md` for the factor list
2. `docs/agent-context/factor-interpretation.md` for `gamma` versus `lambda`

Current deep dossiers exist for the recurring top factors in the main results:

- `pead.md`
- `peadb.md`
- `ivol.md`
- `credit.md`
- `ysp.md`

These dossiers are meant to answer questions like:

- "Fully explain PEAD."
- "Why is IVOL important in this paper?"
- "What does high posterior probability mean for CREDIT?"
- "Why does YSP show up if it is nontraded?"

For appendix robustness tied to PEAD and PEADB, also use:
