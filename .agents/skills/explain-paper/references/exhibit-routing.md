# Exhibit Routing

Use this when the user asks for an exhibit, manuscript claim, or equation.

Routing order:

1. `docs/agent-context/factor-interpretation.md` for gamma-versus-lambda questions
2. `docs/agent-context/factors/` for top-factor explanations
3. Relevant exhibit dossier in `docs/agent-context/exhibits/`
4. `docs/manifests/exhibits.csv` for executable code path and output path
5. `docs/manifests/manuscript_exhibits.csv` for full-paper coverage questions
6. `docs/manifests/paper_claims.csv` for headline claim-to-evidence questions
7. `docs/paper/co-pricing-factor-zoo.ai-optimized.md` for exact equation or appendix wording
8. Source code only after the higher-level mapping is clear

Do not treat paper-only IA exhibits as executable repo outputs.
