# Exhibit Routing

Use this when the user asks for an exhibit, manuscript claim, or equation.

Routing order:

1. Relevant exhibit dossier in `docs/agent-context/exhibits/`
2. `docs/manifests/exhibits.csv` for executable code path and output path
3. `docs/manifests/manuscript_exhibits.csv` for full-paper coverage questions
4. `docs/manifests/paper_claims.csv` for headline claim-to-evidence questions
5. `docs/paper/co-pricing-factor-zoo.ai-optimized.md` for exact equation or appendix wording
6. Source code only after the higher-level mapping is clear

Do not treat paper-only IA exhibits as executable repo outputs.
